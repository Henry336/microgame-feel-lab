# Roadmap

## Current Direction

Build a compact, first-screen playable game-feel sandbox that makes common feel parameters tactile within five minutes.

## Near-Term Priorities

- Add a clearer challenge route with timing gates so presets can be compared against a consistent task.
- Add before/after snapshots so users can A/B a custom feel profile against a preset.
- Improve mobile play controls once desktop tuning is stronger.
- Add richer impact feedback with simple sound synthesis that remains optional.

## Deferred Ideas

- GIF/video capture.
- Sharing links with compressed config state.
- A Phaser or Pixi migration if the canvas scene becomes complex enough to justify an engine.

## Rejected For Now

- Accounts, multiplayer, community preset galleries, and generic game-engine features.
- Marketing pages before the lab itself feels good.
